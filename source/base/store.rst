
仓库 
---------------------

属性
========================
* 名称 - 仓库名称,必须输入
* 成本核算方法 - 当前未用
* 隶属部门 - 必须输入
* 最高限额 - 当前未用
* 使用调拨价 - 当前未用
* 位置
* 备注

搜索 仓库
========================
输入仓库名称或名称拼音的首字母,点击搜索按钮

新建 仓库
========================
1. 点击新建按钮
2. 输入仓库属性
3. 点击保存按钮

编辑 仓库
========================
1. 选择要要编辑的仓库
2. 双击或点击编辑按钮
3. 修改仓库属性
4. 点击保存按钮

删除 仓库
========================
*已使用的仓库不能删除*

1. 选择要删除的仓库
2. 点击删除按钮
3. 确认
